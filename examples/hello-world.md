# Hello World Course

Welcome to your first video course! This minimal example demonstrates the basic structure of a Course Creator course.

## Introduction

Hello and welcome to this demonstration course. Course Creator converts markdown files into professional video courses with AI narration.

## Getting Started

To create a course:
1. Write content in markdown format
2. Use headings for lesson structure
3. Include code examples and explanations
4. Generate video course with AI
