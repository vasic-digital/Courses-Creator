# Quick Demo Course

This is a single-lesson demonstration of Course Creator's capabilities. Perfect for testing the system quickly.

## Course Creator Demo

Course Creator is a powerful tool that transforms markdown scripts into engaging video courses. It features:

- AI-powered text-to-speech narration
- Professional video assembly with backgrounds
- Cross-platform desktop and mobile players
- REST API for integration
- Multiple voice options and quality settings

The system processes markdown files, extracts lessons, generates audio narration, and creates video content with text overlays and visual enhancements.

Try generating this course to see the complete pipeline in action!
